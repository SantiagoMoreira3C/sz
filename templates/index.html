Bienvenido

Descripción de un sistema enfocado a medir la frecuencia cardiaca y la saturación de oxígeno, utilizando machine learning para determinar resultados futuros y detectar enfermedades. Programado con las siguientes librerías de Python: Flask, Mongo, Matplotlib.pyplot, Pandas y Scikit-learn.
